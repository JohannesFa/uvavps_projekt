# Local LLM Evaluation Framework
An automated benchmarking tool built in Python to evaluate the performance and accuracy of local Large Language Models (LLMs) via the Ollama API.
## Project Goals
The goal of this project is to create a streamlined pipeline for testing Small Language Models on local hardware, comparing their reasoning capabilities against structured datasets.
## Tech Stack
* Language: Python
* Inference Engine: Ollama
* Data Handling: Pandas
* Format: CSV-based dataset evaluation
